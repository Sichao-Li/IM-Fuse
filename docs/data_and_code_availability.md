# Data And Code Availability

- **Data:** Stable sample identifiers, target-specific metadata, exact split
  assignments for seeds 0-4, and checksums are included in this repository.
  The larger `mp_total.csv`, CIF collection, atom initializer, and processed
  modality caches are distributed with the archival data release
  (`Zenodo DOI: pending`). Externally sourced Materials Project records remain
  subject to their original terms.
- **Code:** IM-Fuse v1.0.0 is available at
  <https://github.com/Sichao-Li/IM-Fuse> and will be archived on Zenodo
  (`software DOI: pending`). The tagged archive is the version of record.
- **Additional information:** Installation, data preparation, full experiment
  commands, output locations, and adaptation guidance are provided in this
  repository. Pretrained ALIGNN is an optional external dependency and is
  identified separately from models trained within IM-Fuse.

Suggested repository citation before DOI assignment:

> Li, S. et al. IM-Fuse: Interpretable Multimodal Fusion for Battery Materials,
> version 1.0.0 (2026), https://github.com/Sichao-Li/IM-Fuse.
