# IM-Fuse Public Repository Manifest

This branch is the clean public release for **IM-Fuse: Interpretable
Multimodal Fusion for Battery Materials**. It is meant to be small enough for
GitHub while preserving the reusable framework code, interpretation utilities,
split definitions, and scripts needed for review and reproduction. Generated
`results/` and `figures/` artifacts are intentionally excluded so users can
reproduce them locally.

## Included

- `src/battery_fusion/`: package code for data handling, feature builders,
  neural models, fusion models, baselines, revision experiments, and
  explanation utilities.
- `src/battery_fusion/cli.py`: the public `imfuse` command-line facade.
- `src/battery_fusion/data/foundation.py`: reproducible staging path from
  external `mp_total.csv`/CIF sources to labels, splits, and modality caches.
- `scripts/reproduce_publication.sh`: one-command final-publication runner.
- `scripts/run_final_publication_gpu.sh`: configurable publication runner used
  by `scripts/reproduce_publication.sh`.
- `configs/`: retained experiment config and publication defaults.
- `docs/`: public runbooks, result index, architecture notes, and data
  manifest.
- `tests/`: lightweight unit tests that do not require large training data.
- `data/labels/` and `data/splits/`: compact label and split CSVs for the
  retained raw-target publication scope.

## Excluded

- Raw source data: `data/raw/mp_total.csv`, CIF files, and atom initializers.
- Processed tensor caches: `data/processed/`.
- Generated `results/` and `figures/` folders.
- Checkpoints, learned model binaries, raw prediction CSVs, embeddings, and
  cached pretrained readouts.
- Local virtual environments, logs, LMDB folders, Python caches, and archived
  exploratory runs.
- Exploratory standardized-target, gated-fusion, M3GNet, CrabNet/Roost, and
  local crystal-rendering trial outputs.

## External Artifacts Needed For Full Reruns

Place these paths before running the full publication pipeline:

```text
data/raw/mp_total.csv
data/raw/cifs/*.cif
data/raw/atom_init.json
data/processed/legacy_rdf_split_seed_42/
```

Alternatively, rebuild them from an external release:

```bash
imfuse prepare-data --target-col average_voltage --mp-total-url "$MP_TOTAL_URL" --cif-archive-url "$CIF_ARCHIVE_URL" --seeds 0 1 2 3 4 --preprocess
```

The random-split and halide-holdout split CSVs are already included. Summary
tables, figures, raw prediction CSVs, and checkpoints can be regenerated by
`scripts/run_final_publication_gpu.sh`.
